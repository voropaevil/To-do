Сервис To-do - одностраничное приложение списка дел

Самое главное, что реализованы все пункты для выполнения проектной работы, а также:
1) localStorage для персистентности данных
2) Разбиение на небольшие функции с одной ответственностью
3) Вроде корректное использование методов DOM
4) Проверка пустого ввода

Функционал:
Просмотр задач — при открытии страницы отображается список дел, загруженный из локального хранилища или предустановленный по умолчанию.
Добавление — новая задача добавляется в начало списка через форму ввода.
Удаление — каждая задача может быть удалена нажатием на кнопку с корзиной.
Копирование — создаёт точную копию задачи и помещает её на первое место.
Редактирование — нажмите на кнопку с карандашом, чтобы изменить текст задачи. Изменения сохраняются автоматически при потере фокуса.
